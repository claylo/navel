# Colophon

This is an unofficial reference compiled from the [Claude Code documentation](https://code.claude.com/docs) published by [Anthropic](https://anthropic.com).

It is not affiliated with or endorsed by Anthropic.

## Source

Documentation was fetched from `code.claude.com/docs` using the site's `llms.txt` index. Navigation structure was extracted from the site's embedded configuration. All content is copyright Anthropic, PBC.

## Production

Built with [navel](https://github.com/claylo/navel), a toolkit for tracking Claude Code releases, commands, hooks, and documentation across versions.

The rendering pipeline uses [@mintlify/mdx](https://mintlify.com) to process MDX source files, producing static HTML styled to approximate the official documentation site. External images are localized for offline use.
